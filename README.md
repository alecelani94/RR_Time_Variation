# Bilinear TVP-VAR

Bilinear Time-Varying Parameter Vector Autoregression.

## Project Structure

```
Bilinear_TVP_VAR/
├── Functions/      % Utility and helper functions
├── Models/         % Model estimation routines
├── Data/           % Input datasets
├── Output/         % Results, figures, and tables
├── Simulations/    % Monte Carlo simulations
└── main.m          % Main script (entry point)
```

## Requirements

- MATLAB R2020b or later

## Usage

```matlab
% Run the main script
main
```
